# AlefLang Docs

Public documentation for Alef, an AI-native programming language with a native
runtime, built-in HTTP, database, AI, MCP, media, and agent-facing stdlib
surfaces.

This repo is intentionally GitBook-friendly:

- `SUMMARY.md` defines the navigation.
- `docs/*.md` pages are plain Markdown.
- No dependency install is required to read or publish the content.

## Start Here

Read:

1. [Syntax Guide](docs/syntax.md)
2. [Tutorial: Build a Native API](docs/tutorial-native-api.md)
3. [AI-Native Tutorial](docs/tutorial-ai-native.md)
4. [Native Runtime](docs/native-runtime.md)

## Local Preview

Any Markdown viewer works. For GitBook, import the repository and use
`SUMMARY.md` as the table of contents.
