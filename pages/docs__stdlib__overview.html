Standard Library Overview

Alef is a programming language, and the standard library is part of that language experience.

Common surfaces:

  • std.http for native HTTP servers and clients
  • std.json / json_encode / json_decode for structured data
  • std.ai for provider calls
  • std.db for database-backed examples
  • std.cache for small stateful helpers
  • std.view and std.http.html for rendered HTML examples

The stdlib should be documented like a language stdlib: module by module, function by function, with runnable examples.
