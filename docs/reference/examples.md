# Examples

## TicketDesk UI

Repository: `alef-ticketdesk-ui`

What it demonstrates:

- multi-workspace ticket board
- server-rendered UI
- status movement
- assignment
- activity feed
- JSON APIs
- smoke test state verification

Run:

```bash
ALEF_BIN=/path/to/alef bash scripts/smoke.sh
```
